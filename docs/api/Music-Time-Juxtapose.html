Prerequisites

type Transformable a = (Stretchable a, Delayable a)

This pseudo-class gathers the restrictions needed to implement position a value at any point and duration in time.

Juxtaposing values

follow :: (HasOnset a, HasOffset a, Delayable a) => a -> a -> a

a `follow` b moves score b so that its onset is at the offset of score a and returns the moved score.

precede :: (HasOnset a, HasOffset a, Delayable a) => a -> a -> a

a `precede` b moves score a so that its offset is at the onset of score b and returns the moved score.
