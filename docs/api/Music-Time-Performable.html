Description

Provides performable values. To perform a score is to render it as a list of time-duration-value triplets.

The compose function captures the fact that a value that is a Monoid, Transformable and Pointed can be reconstructed from a performance, i.e. compose is the inverse of perform.

HasPoint class

The HasPoint class serves only to lift the Pointed class into unary kinds. We could have used Applicative or Monad here, but as we only care about pure and return, Pointed makes more sense.

Recall that: 

 pure = return = point

type HasPoint a = (a ~ Container a (Event a), Pointed (Container a))

point' :: HasPoint a => Event a -> a

Equivalent to point, pure and return.

Composable values

class (Monoid' a, Transformable a, HasPoint a) => Composable a where

Composable values.

As one might expect 

 compose . perform = id
 perform . compose = id

That is compose is the inverse of perform.

The methods in this class have a default implementation in terms of its superclasses, but are provided so that implementors may provide more efficient implementations.
